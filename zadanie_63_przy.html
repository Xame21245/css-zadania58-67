__________początek teorii :first-child _______
Użycie :first-child powoduje że pierwszy akapit (znacznik <p>, może być wiele akapitów
<p> ) w bloku (np. <DIV>), będzie inaczej formatowanu niż pozostałe akapity w tym samym
bloku ( dla całego dokumentu, czyli dla wszystkich DIV).


Dwa pliki
  • arkusz styli
  • dokument
współpracują ze sobą.

Przykład poniższy (arkusz styli oraz dokument) realizują:
      1. dla każdego akapitu ustawia czcionkę 14-punktową.
      2. pierwszy akapit w bloku DIV będzie wyglądał nieco inaczej, gdyż użyta czcionka
          będzie pochyła i o wysokości 10 punktów.


arkusz styli

p
{
font-size:14pt;
}
p:first-child
{
font-size:10pt;
font-styl:italic;
}

Dokument

<div>
<p>Pierwszy akapit w ramach pierwszego bloku DIV</p>
<p>Drugi akapit w ramach pierwszego bloku DIV</p>
<p>Trzeci akapit w ramach pierwszego bloku DIV tutaj wpisz Twoje nazwisko</p>
</div>
<div>
<p>Pierwszy akapit w ramach drugiego bloku DIV</p>
<p>Drugi akapit w ramach drugiego bloku DIV</p>
<p>Trzeci akapit w ramach drugiego bloku DIV-> tutaj wpisz Twoje imię</p>
</div>
